## Installation

The package can be installed
by adding `tello` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:tello, "~> 0.3.0"}
  ]
end
```

The docs can
be found at <https://hexdocs.pm/tello>.
