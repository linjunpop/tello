# Tello

An unofficial [Ryze Tello](https://www.ryzerobotics.com/tello-edu) SDK.

## Installation

The package can be installed
by adding `tello` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:tello, "~> 0.1.0"}
  ]
end
```

The docs can
be found at <https://hexdocs.pm/tello>.
